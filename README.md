# Dashboard de Rendimiento – Alan por el Mundo

Este proyecto es un dashboard responsive construido con HTML y Tailwind CSS v4 para analizar el rendimiento del creador de contenido Alan por el Mundo en sus plataformas digitales.

## Tecnologías utilizadas
- HTML
- Tailwind CSS v4 (CDN oficial)
- Mobile-first design
- Sin JavaScript ni frameworks (React, Vue, Angular)

## Estructura del Dashboard

### 1️⃣ Bloque superior – KPIs principales
Incluye métricas de:
- Ingresos (€)
- Engagement
- Conversiones
- Rendimiento general
- Mini resumen narrativo con insights

### 2️⃣ Bloque intermedio – Drivers
Incluye:
- Funnel de ventas
- Rendimiento por plataforma (YouTube, Instagram, TikTok)
- Rendimiento por producto
- Actividad por plataforma
- Barras de progreso visuales

### 3️⃣ Bloque inferior – Detalles operacionales
Incluye:
- Tabla de productos (precio, ventas, comisiones, ROI)
- Tabla de plataformas (alcance, engagement, conversiones)
- Tabla de campañas
- Alertas visuales
- Listados “Top”

## Características visuales destacadas
- Hero KPI con degradado profesional
- Glassmorphism en tarjetas
- Barras de progreso sin JavaScript
- Iconos y emojis para mejorar la lectura
- Alertas visuales con colores fuertes
- Sección de “Insights del día”
- Diseño mobile-first con breakpoints para móvil, tablet y escritorio
